# Лабораторна робота: TCP-клієнт-сервер + Docker

## Функціонал
- Сервер постійно працює і обробляє багато клієнтів
- Обчислює середню кількість кроків гіпотези Колатца для 1..N
- Клієнт читає параметри з змінних оточення
- Два контейнери спілкуються через Docker-мережу
- Healthcheck + depends_on з умовою service_healthy

## Запуск
```bash
docker compose up --build
